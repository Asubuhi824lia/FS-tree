## FS-tree

FS-tree - web-application for display FS tree (received from the server in the form of a `json` file) made on `React`.

#### Requirements
- Node.js

#### Install

```
$ git clone https://github.com/Asubuhi824lia/FS-tree.git
$ cd FS-tree
$ npm init
```

#### Development

```
$ npm start
```
