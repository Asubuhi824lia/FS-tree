## FS-tree (next.js)

FS-tree - web-application for display FS tree (received from the server in the form of a `json` file) made on `Nextjs`.

#### Requirements
- Node.js

#### Install

```
$ git clone https://github.com/Asubuhi824lia/FS-tree.git
$ cd FS-tree
$ git checkout next.js
$ npm init
```

#### Development

```
$ npm run dev
```
